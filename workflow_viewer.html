Workflow: internationalSolar_cold_email_outreach_v1

This workflow processes international solar company leads through webhook trigger

Workflow Overview:

  1. webhookTrigger - Receives incoming data
  2. theirStackSearch - HTTP request to search company data
  3. splitCompanies - Batch process companies
  4. hunterSearch - Find contacts via Hunter API
  5. combineData - Merge company and contact data
  6. checkContacts - Verify if contacts found
  7. splitContacts - Batch process contacts
  8. classifyContact - Categorize HR vs Tech leads
  9. addHrLead - Add HR leads to system
  10. addTechLead - Add Tech leads to system
  11. webhookResponse - Send response back
  12. companyResearch - AI-powered company analysis

Key Configuration Points:

  • HTTP nodes likely need API credentials for TheStack and Hunter
  • AI Transform node needs configuration for company research
  • Webhook URL needs to be configured for trigger
